# Changes from forked repo
* Topologies: the original repo (as well as the original game) only plays on a torus
* Friendly fire: added a game setting where your bullets can kill you (just like in real life)
* Brakes: to make controls slightly less annoying, added a braking button (&#8595;) that behaves exactly like thrusting in the direction opposite your motion.
* Misc: there are a few visual changes/changes in mechanics (e.g. bullets also obey the topology, ship is filled in, etc.)
